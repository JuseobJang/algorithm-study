## Baekjoon 2193 : 이친수

### 논리순서

1. dp[i][0] , dp[i][1] 은 각각 수의 마지막이 0 또는 1로 끝났을 떄의 갯수이다.
2. 마지막이 0 으로 끝났을 경우, 이전 자리수의 마지막이 0또는 1로 끝났을 수 있다. 
3. 마지막이 1 로 끝났을 경우, 이전 자리수의 마지막이 무조건 0으로 끝난것이다.
4. 따라서 각각 점화식의 나타내면 아래와 같다.
   - dp[i][0] = dp[i-1][0] + dp[i-1][1]
   - dp[i][1] = dp[i-1][0]

```java
dp[1][0] = 0;
dp[1][1] = 1;
for (int i = 2; i <= n; i++) {
    dp[i][0] = dp[i - 1][0] + dp[i - 1][1];
    dp[i][1] = dp[i - 1][0];
}

```
