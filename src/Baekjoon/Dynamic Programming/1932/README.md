# Baekjoon 1932 : 정수 삼각형

## Solution Logic

1. 삼각형 형태이기 때문에 각 층의 첫번째와 마지막은 각각 이전층의 첫번째와 마지막층만 참고하면 된다.

2. 따라서 각 층의 첫번쨰와 마지막 수를 제외하고는
    dp[i][j] = max(dp[i-1][j-1] , dp[i-1][j]) + tri[i][j] 로 둘수 있다.

3. 각 층의 첫번째(j==0)

    dp[i][j] = dp[i - 1][j] + tri[i][j]

4. 각 층의 마지막(i==j)

    dp[i][j] = dp[i - 1][j - 1] + tri[i][j]
