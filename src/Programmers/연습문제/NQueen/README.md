# Programmers

## 연습문제 : N-Queen

### Soulution Logic

1. BackTraking 알고리즘 사용

2. 그리디하게 map을 조회하면서 가능한지 비교하면 시간초과

3. 한 행에는 하나의 퀸만 들어갈 수 있음

4. 아래 행에서 윗 행들의 퀸의 자리에 따라서 퀸의 자리를 정함

5. 불가능한 자리 col[j] - n + j == i || col[j] == i || col[j] + n - j == i (i : 퀸을 넣을 수 있는지 확인할 자리, j : 이미 놓여진 퀸의 행, n : 현재 퀸의 행)

6. 위조건은 각각 좌대각선, 아래, 우대각선 방향을 나타냄

7. 해당 조건을 만족하지 않는 부분에대해서만 재귀함수를 진행함.
